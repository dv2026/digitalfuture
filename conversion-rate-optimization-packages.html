What's the difference between A/B testing and multivariate testing?

A/B testing compares one version of a page against another (Version A vs Version B), while multivariate testing checks multiple element combinations at the same time (for example, headline + button + image). A/B testing is ideal for most Conversion Rate Optimization Packages, while multivariate testing is used on high-traffic pages with many elements to refine.
